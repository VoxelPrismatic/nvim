# nvim
My custom neovim flavor

# install
```sh
rm -rdf ~/.config/nvim
rm -rdf ~/.local/share/nvim
git clone https://github.com/VoxelPrismatic/nvim ~/.config/nvim
```

# update
```
cd ~/.config/nvim
git reset --hard
git pull
```
